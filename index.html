These animations are created using GlowScript on trinket.io. To create an animation, select GlowScript in the dropdown menu for a new Trinket and code away! These can also be run on your computer using Python with the VPython package. Documentation for using VPython is available here.

Jupyter notebooks (.ipynb) are another browser-based interface to run Python code. Google Colaboratory lets you create these Python notebooks and run them on their machine. Since they are saved on your Google Drive, they can be shared very easily. Unfortunately, VPython animations may be difficult to run on Google Colaboratory. But this tool can still be used for arithmatic calculations using numpy and making scatter plots with matplotlib. To get started, download this .ipynb file and upload it to your Google Drive. On Google Drive, select 'Open with → Connect more apps', and install Colaboratory on your account. This will let you open and run Jupyter Notebooks directly on your browser.
