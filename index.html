Physics Simulations using GlowScript and Python

These animations are created using GlowScript on trinket.io. To create an animation, select GlowScript in the dropdown menu for a new Trinket and code away! These can also be run on your computer using Python with the VPython package. Documentation for using VPython is available here.

Why do this?

  • Develop physics intuition
  • Learn physics modelling
  • Exposure to coding
  • Useful teaching tool
